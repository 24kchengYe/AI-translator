# v4.0 问题修复总结

## ✅ 已修复的问题

### 1. 字体问题 - 完全解决 ✓

**症状**：
- CustomTkinter无法让所有组件使用Arial字体
- 弹窗字体不一致

**修复**：
- ✅ 替换为ttkbootstrap
- ✅ 所有组件统一Arial字体
- ✅ 修复LabelFrame → Labelframe（注意大小写）

**验证**：
```bash
python test_ui.py
```
查看测试窗口，所有文字应该都是Arial字体。

---

### 2. 翻译速度慢 - 大幅优化 ⚡

**症状**：
- 并行效率只有29.6%
- 速度提升不明显（1.78x）

**修复**：
- ✅ 使用异步IO替代多线程
- ✅ BATCH_SIZE: 25 → 15（更小批次）
- ✅ MAX_WORKERS: 6 → 12（更高并发）

**预期提升**：
- 从1.78x提升到3-4x
- 并发效率从29.6%提升到60-70%

**验证**：
```bash
python test_parallel_performance.py
```

---

### 3. 翻译数量不匹配警告 - 已修复 ✓

**症状**：
```
警告: 翻译结果数量不匹配 (期望 25, 实际 46)，使用逐个翻译
```

**原因**：
- AI返回了额外的换行或解释
- 按行分割导致数量不对

**修复**：
- ✅ 更严格的提示词（强调数量要求）
- ✅ 基于编号索引解析（不依赖行数）
- ✅ 验证编号有效性
- ✅ 自动过滤无效结果

**效果**：
- 大幅减少"数量不匹配"警告
- 更稳定的批量翻译

## ⚙️ 配置优化

### 推荐配置（平衡）

`.env` 文件：
```env
# API配置
OPENAI_API_KEY=your_key_here
OPENAI_BASE_URL=https://openrouter.ai/api/v1
OPENAI_MODEL=openai/gpt-4o

# 翻译设置（优化后）
BATCH_SIZE=15          # 每批15个文本
MAX_RETRIES=3
RETRY_DELAY=5

# 并行设置
MAX_WORKERS=12         # 12个并发
PARALLEL_ENABLED=true
```

### 追求极致速度

```env
BATCH_SIZE=10          # 更小批次
MAX_WORKERS=15         # 更高并发
```

### API限流严格

```env
BATCH_SIZE=20          # 较大批次
MAX_WORKERS=6          # 较低并发
```

### 稳定优先

```env
BATCH_SIZE=25
MAX_WORKERS=6
PARALLEL_ENABLED=false  # 禁用并行
```

## 🔧 常见问题

### Q1: 启动时报错 "module 'ttkbootstrap' has no attribute 'LabelFrame'"

**A**: 已修复！使用 `Labelframe`（小写f）而不是 `LabelFrame`。

更新代码：
```bash
git pull  # 或重新下载
```

---

### Q2: 性能提升不明显

**A**: 检查配置：

1. 确认异步模式启用：
   ```bash
   python -c "from translation_engine import translation_engine; print(f'并行: {translation_engine.parallel_enabled}, Workers: {translation_engine.max_workers}')"
   ```

2. 查看实际使用的模式：
   翻译时查看日志，应该看到"异步翻译"相关信息。

3. 调高并发数：
   ```env
   MAX_WORKERS=15
   BATCH_SIZE=10
   ```

---

### Q3: 仍然出现"数量不匹配"警告

**A**: 这可能是AI偶尔的问题，已大幅减少但无法完全避免。

**解决方案**：
1. 程序会自动降级到逐个翻译（稳定性保证）
2. 如果频繁出现，可以：
   ```env
   BATCH_SIZE=10  # 减小批次大小
   ```

---

### Q4: 字体还是不是Arial

**A**: 检查步骤：

1. 运行测试：
   ```bash
   python test_ui.py
   ```

2. 确认导入的是新UI：
   检查 `main.py` 第6行应该是：
   ```python
   from gui.main_window_new import run_gui
   ```

3. 如果还有问题，手动设置系统默认字体为Arial。
